# 排序

## 定义

### 排序的一般定义

排序是计算机内经常进行的一种操作， 其目的是将一组“无序＂ 的数据元素调整为有序＂ 的数据元素。

例如：将下列关键字序列

52,49,80,36, 14,58,61,23,97, 75

调整为：

14,23,36,49,52,58,61 ,75,80,97

### 排序的数学定义

## 排序的稳定性

如果在序列中有两个数据元素 R[i] 和 R[j]，它们的关键字满足 k[i] == k[j]，且在排序之前，对象 R[i] 排在 R[j] 前面；若在排序之后，对象 R[i] 仍在对象 R[j] 的前面，则称这个排序方法是稳定的，否则称这个排序方法是不稳定的。

## 多关键字排序

多关键字排序

当排序时需要比较的关键字多于一个时：

排序结果首先按关键字1进行排序

当关键字1相同时，按关键字2进行排序

...

当关键字n - 1相同时，按关键字n进行排序

## 排序的评判标准

**时间性能**

关键性能差异体现在比较和交换的次数

**辅助存储空间**

为完成排序操作需要的额外存储空间

必要时可采用"空间换时间"策略

**算法的实现复杂性**

过于复杂的排序算法可能影响可读性和可维护性

## 选择排序

选择排序是一种不稳定的排序，时间复杂度为O(n^2)

### 选择排序的基本思想

每次（例如第 i 次，其中 i = 0, 1, …, n - 2）从后面 n - i 个待排序的数据元素中选出关键字最小的元素，作为有序元素序列的第 i 个元素。

## 插入排序

插入排序是一种稳定的排序，时间复杂度为O(n^2)

### 插入排序的基本思想

当插入第 i 个数据元素时（其中 i ≥ 1），前面的元素 V[0], V[1], ..., V[i - 1] 已经排好序。这时：

1. 用 V[i] 的关键字与 V[i - 1], V[i - 2], ..., V[0] 的关键字从后向前进行比较

2. 找到合适位置后，将 V[i] 插入

3. 原来位置上的元素向后顺次移动

```cpp
{
```

## 冒泡排序

冒泡排序是一种稳定的排序，时间复杂度为O(n^2)

### 冒泡排序的基本思想

每次从前向后进行（假设为第i次），j=n-1,n-2,…,i,两两比较V[j-1]和V[j]的值，

如果发生逆序，则交换V[j-1]和V[j]

希尔排序是不稳定的，时间复杂度为O(n^3/2)

将待排序序列划分为若干子序列，在每个子序列内进行插入排序通过逐步缩小子序列间隔，使整个序列趋向基本有序，最后对整个序列进行一次插入排序

```cpp
template<typename T>
static void Shell(T array[], int len, bool min2max = true)
{
    int d = len;

    do
    {
        d = d / 3 + 1;

        for (int i = d; i < len; i += d)
        {
            int k = i;
            T e = array[i];

            for (int j = i - d; (j >= 0) && (min2max ? (array[j] > e) : (array[j] < e)); j -= d)
            {
                array[j + d] = array[j];
                k = j;
            }

            if (k != i)
            {
                array[k] = e;
            }
        }
    } while (d > 1);
}

void sj48()
{
```

## 快速排序

快速排序的时间复杂度为O(n*logn) ， 是—种不稳定的排序法

### 快速排序的基本思想

任取序列中的某个数据元素作为**基准**，将整个序列划分为左右两个子序列：

**左侧子序列**：所有元素都**小于或等于**基准元素

**右侧子序列**：所有元素都**大于**基准元素

**基准元素**位于两个子序列中间

分别对两个子序列**重复**进行划分，直到所有元素都排列在相应位置上

### 排序类快速排序代码

以下代码比较像挖空法，所以在末尾加上了array[begin] = pv;但也许并不需要这句，只是以把该元素挖出来不去理会这个元素的存在的思想去写而加上了最后把挖的基准值填回去的这句代码。

```cpp
template<typename T>
static int Partition(T array[], int begin, int end, bool min2max)
{
    T pv = array[begin];

    while (begin < end)
    {
        while ((begin < end) && (min2max ? (array[end] > pv) : (array[end] < pv)))
        {
            end--;
        }
        Swap(array[begin], array[end]);

        while ((begin < end) && (min2max ? (array[begin] <= pv) : (array[begin] >= pv)))
        {
            begin++;
        }

        Swap(array[begin], array[end]);
    }

    array[begin] = pv;

    return begin;
}

template<typename T>
static void Quick(T array[], int begin, int end, bool min2max)
{
    if (begin < end)
    {
        int pivot = Partition(array, begin, end, min2max);

        Quick(array, begin, pivot - 1, min2max);
        Quick(array, pivot + 1, end, min2max);
    }
}
template<typename T>
static void Quick(T array[], int len, bool min2max = true)
{
    Quick(array, 0, len - 1, min2max);
}


```

## 归并排序

也采用“分治思想”，先进行序列划分，再进行元素的有序合并。

### 一、基本思想

归并排序采用分治法（Divide and Conquer），其核心步骤如下：

分治策略

将原始数组 A[0:n−1] 分成两个大小大致相等的子数组：

前半部分： A[0:⌊n/2⌋]

后半部分： A[⌊n/2⌋+1:n−1]

递归终止条件

当子数组长度为 1 时，自然有序，无需进一步拆分。

合并（Merge）过程

对两个已排序的子数组进行线性时间合并，生成一个有序的大数组。

递归实现：不断二分数组，直到单元素，再逐层合并。

### 二、稳定性分析

归并排序的稳定性：稳定

不直接交换元素，仅通过比较决定合并顺序。

合并时优先选择前半部分数组的元素（若前后元素相等）。

### 三、时间复杂度

总时间复杂度： O(nlogn)

递推关系：

解析：

递归拆分：将问题分解为两个 n/2 规模的子问题，耗时 2T(n/2)。

合并操作：合并两个已排序子数组，耗时 O(n)。

递推求解：

令 x=logn（递归深度），则：

T(n)=2xT(1)+x⋅O(n)=n⋅O(1)+logn⋅O(n)=O(nlogn)

简单来看递归层数logn * 归的时间复杂度O(n)即为结果

### 四、空间复杂度

缓冲区空间：

合并过程中需要临时存储两个子数组的合并结果。

最大占用：一次合并分配 O(n) 的缓冲区空间。

递归栈空间：

递归调用栈深度为logn，每层栈帧占用常数空间。

总栈空间：O(logn)。

总空间复杂度： O(n)（由缓冲区主导）。

```cpp
void MergeSort(int arr[], int l, int m, int r, int* p)
{
    int i = l;
    int j = m + 1;
    int index = 0;

    while (i <= m && j <= r)
    {
        if (arr[i] <= arr[j])
        {
            p[index++] = arr[i++];
        }

        if (arr[j] < arr[i])
        {
            p[index++] = arr[j++];
        }
    }

    while (i <= m)
    {
        p[index++] = arr[i++];
    }

    while (j <= r)
    {
        p[index++] = arr[j++];
    }

    for (i = l, j = 0; i <= r; i++, j++)
    {
        arr[i] = p[j];
    }
}
void MergeSort(int arr[], int begin, int end, int* p)
{
    if (begin >= end)
    {
        return;
    }

    int mid = (begin + end) / 2;
    MergeSort(arr, begin, mid, p);
    MergeSort(arr, mid + 1, end, p);

    MergeSort(arr, begin, mid , end, p);
}

void MergeSort(int arr[], int size)
{
    int* p = new int[size];

    MergeSort(arr, 0, size - 1, p);
}

int main()
{
    int arr[10];
    srand(time(nullptr));

    for (int i = 0; i < 10; i++)
    {
        arr[i] = rand() % 100 + 1;
    }

    for (int v : arr)
    {
        std::cout << v << " ";
    }
    std::cout << std::endl;

    MergeSort(arr, 10);

    for (int v : arr)
    {
        std::cout << v << " ";
    }
    std::cout << std::endl;
```

### 二叉堆

满足 **0 <= i <= (n-1)/2**，n代表最后一个元素的下标。

**如果arr[i] <= arr[2 * i+1] && arr[i] <= arr[2 * i+2]，就是小根堆**

**如果arr[i] >= arr[2 * i+1] && arr[i] >= arr[2 * i+2]，就是大根堆**

#### PriorityQueue代码

push插入一个元素 到优先级队列中，并通过 上浮（siftUp） 维护堆的性质

pop**删除堆顶元素**（优先级最高），并将最后一个元素移动到堆顶，通过 **下沉（siftDown）** 维护堆的性质。

```cpp
class PriorityQueue
{
    private:
        using Comp = function<bool(int, int)>;

        Comp m_comp;
        int* m_queue;
        int m_size;
        int m_capacity;

    public:
        PriorityQueue(int cap = 20,Comp comp = greater<int>())
        :m_capacity(cap),m_size(0), m_comp(comp)
        {
            m_queue = new int[20];
        }
        PriorityQueue(Comp comp)
        :m_capacity(20), m_size(0), m_comp(comp)
        {
            m_queue = new int[20];
        }

        ~PriorityQueue()
        {
            delete[]m_queue;

            m_queue = nullptr;
        }

        void push(int val)
        {
            if (m_size == m_capacity)
            {
                int* mem = new int[2*m_capacity];

                memcpy(mem, m_queue, m_capacity * sizeof(*m_queue));

                int* toDel = m_queue;
                m_queue = mem;
                m_capacity *= 2;
                delete[]toDel;
            }

            if (m_size == 0)
            {
                m_queue[m_size] = val;
            }
            else
            {
                siftUp(m_size, val);
            }

            m_size++;
        }

        void pop()
        {
            if (m_size == 0)
            {
                throw "No elment in container";
            }

            m_size--;

            if (m_size > 0)
            siftDown(0, m_queue[m_size]);
        }

        bool empty() const{ return m_size == 0; }

        int top()const
        {
            if (m_size == 0)
            throw "container is emtpy!";

            return m_queue[0];
        }

        int size()const { return m_size; }

    private:
        void siftUp(int i, int val)
        {
            while (i > 0)
            {
                int father = (i - 1) / 2;

                if (m_comp(val, m_queue[father]))
                {
                    m_queue[i] = m_queue[father];

                    i = father;
                }
                else
                {
                    break;
                }
            }

            m_queue[i] = val;
        }
        void siftDown(int i, int val)
        {
            while (i < m_size / 2)
            {
                int child = 2 * i + 1;
                if (child+1<m_size && m_comp(m_queue[child + 1], m_queue[child]))
                {
                    child += 1;
                }

                if (m_comp(m_queue[child], val))
                {
                    m_queue[i] = m_queue[child];

                    i = child;
                }
                else
                {
                    break;
                }
            }

            m_queue[i] = val;
        }
};


```

```cpp
void siftDown(int arr[],int i, int size)
{
    int val = arr[i];

    // i下沉不能超过最后一个有孩子的节点
    while (i < size / 2)
    {
        int child = 2 * i + 1; // 第i个节点的左孩子
        if (child + 1 < size && arr[child + 1]> arr[child])
        {
            // 如果i节点右孩子的值大于左孩子, child记录右孩子的下标
            child = child + 1;
        }

        if (arr[child]> val)
        {
            arr[i] = arr[child];
            i = child;
        }
        else
        {
            break; // 已经满足堆的性质，提前结束
        }
    }
    arr[i] = val;
}

void heapSort(int arr[],int len)
{
    for (int i = len / 2 - 1; i >= 0; i--)
    {
        siftDown(arr, i, len);
    }

    for (int i = len - 1; i > 0; i--)
    {
        int tmp = arr[0];
        arr[0] = arr[i];
        arr[i] = tmp;

        siftDown(arr, 0, i); // 第三个参数，参与调整的元素的个数
    }
}
```

**为什么堆排序对比快排和归并排序速度慢这么多？**

快排和归并排序，遍历元素时都是按照顺序遍历的，对cpu缓存友好（cpu命中率高）。而堆排序访问元素的时候，是按照父子节点的关系访问的，并不是按照顺序访问的，所以排序过程中，不管是进行元素上浮调整还是下沉调整，对cpu缓存不友好。

堆排序过程中进行下沉调整所作的无效比较过多。选择末尾的元素放到堆顶，因为它本身就小，所以最终下沉到的地方，和末尾位置相差不远，中间做了很多比较，无用功太多。

## 常见面试题

**STL里sort算法用的是什么排序算法**

快速排序算法，插入排序（待排序序列的个数<=32,递归太深转成堆排序

**快速排序的时间复杂度不是稳定的O(n*logn)，最坏情况会变成O(n^2)，怎么解决复杂度恶化问题？**

快速排序的最坏情况（时间复杂度退化为 O(n^2)）发生在每次划分操作中，基准值总是选到当前子数组的最小值或最大值。此时，每次划分只能减少一个元素，导致递归深度为 O(n)，每层的时间复杂度为 O(n)，总时间复杂度为 O(n^2)。从数组的左端、右端和中间位置选取三个元素，取其中间值作为基准

**快速排序递归实现时，递归过深会引发什么问题？怎么解决递归层次过深的问题？**

递归深度过深可能导致栈溢出. 设置递归深度阈值，超过阈值后切换为非递归算法（如堆排序）

**怎么控制递归深度？如果达到递归深度了还没排完序怎么办？**

优先处理小子数组：每次递归先处理较小的子数组，较大的子数组用循环处理，确保递归深度不超过 O(log⁡n)O(logn)。

设置递归深度阈值：若递归深度超过阈值（如 logn），则停止递归，转为迭代处理剩余子数组。

**切换为非递归算法**：当递归深度达到阈值时，用插入排序、堆排序等非递归算法处理剩余部分。
